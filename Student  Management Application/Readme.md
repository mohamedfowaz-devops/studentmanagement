Student Form Management System

A simple web-based Student Form Management System built using HTML, CSS, Bootstrap, and JavaScript. This project allows users to add, view, edit, update, and delete student records along with calculating total marks, average, pass/fail status, and grade.

**Features**
1.Add student details
2.View all student records in table format
3.Edit and update student information
4.Delete student records
5.Automatic total and average calculation
6.Pass/Fail result generation
7.Grade calculation based on marks
8.Responsive UI using Bootstrap
